## Переменные окружения

| Переменная           | Тип    | Описание                                                                       |
|----------------------|--------|--------------------------------------------------------------------------------|
| PORT                 | number | Порт запуска приложения                                                        |
| MONGO_CONNECT        | string | URI для подключения к MongoDB                                                  |
| MONGO_NAME           | string | Название DB в MongoDB                                                          |
